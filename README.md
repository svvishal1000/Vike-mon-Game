# Vike'Mon 🎮

A Pokémon-inspired game built using Python and Pygame.

## Features
- Overworld movement
- Battle system
- Animated intro and menu
- Sound effects
- Custom Vike'mons

## Tech Stack
- Python
- Pygame
- OpenCV
